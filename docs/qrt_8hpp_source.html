22 // This represents the public API for the xacc-enabled
23 // qcor quantum runtime library. The goal here is to provide
24 // and API that compilers can translate high-level
25 // quantum kernel language representations to
26 // (written in openqasm, quil, xasm, etc). The implementation of this API
27 // seeks to build up an xacc::CompositeInstruction with each individual
28 // quantum instruction invocation. Once done, clients can invoke
29 // the submit method to launch the built up program the
30 // user specified backend.
